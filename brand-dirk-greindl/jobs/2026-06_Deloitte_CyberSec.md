# CV Template – Job-Specific Customization

**Use this template for each job application**

---

## 🎯 HOW TO USE

1. **Copy this file:**
   ```bash
   cp jobs/TEMPLATE.md jobs/2026-06_CompanyName_RoleTitle.md
   ```

2. **Read the job description** and note:
   - Key skills mentioned (ISO 26262, ASIL-D, Cybersecurity, etc.)
   - Role level (Manager, Senior, Expert, Consultant)
   - Domain focus (Automotive, Railway, Aerospace)
   - Must-haves vs. nice-to-haves

3. **Edit the CV below** to emphasize relevant experience

4. **Commit to Git:**
   ```bash
   git add jobs/2026-06_CompanyName_RoleTitle.md
   git commit -m "New: CV customized for [Company] [Role]"
   git push
   ```

---

---

## 💡 TIPS

1. **Don't Lie:** Only emphasize real experience; rearrange for relevance
2. **Keywords:** Include exact phrases from job posting (ATS optimization)
3. **Length:** 1–2 pages max (recruiters spend 6 seconds scanning)
4. **Achievements:** Use numbers where possible (team size, budget, timeline)
5. **Keep Originals:** Always save customized versions with company name + date

---

**Save naming convention:**  
`jobs/YYYY-MM_CompanyName_RoleTitle.md`

Examples:
- `jobs/2026-06_Deloitte_CyberSecurity.md`
- `jobs/2026-06_TUV_Auditor.md`
- `jobs/2026-07_Alstom_CertificationLead.md`
